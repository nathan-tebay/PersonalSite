MicrophoneController

2-button USB HID microphone controller with tap-to-toggle and push-to-talk

View on GitHub

Overview

A custom USB HID peripheral built on the nullbits Bit-C PRO (ATmega32U4). Two buttons each support tap-to-toggle and hold-for-push-to-talk. A hardware mode switch selects between keyboard output (Scroll Lock / F24) and gamepad output (buttons 20–21), with LEDs reflecting each mic's live state.

The gamepad HID descriptor is implemented directly via the AVR core's HID.h — no third-party joystick library required.

Stack

Arduino C++ ATmega32U4 USB HID AVR core arduino-cli

Hardware

  • Board: nullbits Bit-C PRO (Arduino Leonardo-compatible)
  • Button 1 — D2, internal pull-up → GND
  • Button 2 — D3, internal pull-up → GND
  • LED 1 — D4 via 220 Ω resistor
  • LED 2 — D5 via 220 Ω resistor
  • Mode switch — D15 to GND = keyboard; floating = gamepad

Button Behaviour

  • Tap — toggle mic on/off (keyboard: key press+release; gamepad: brief button press)
  • Hold ≥ 300 ms — push-to-talk; key/button held until released
  • Mode switch — releases all outputs and resets toggle state
  • Keyboard keys: Scroll Lock (btn 1), F24 (btn 2)
  • Gamepad buttons: 20 and 21 (safely beyond all shipping controller hardware)
