MicrophoneController

2-button USB HID microphone controller with tap-to-toggle and push-to-talk

Problem

In many games and communication apps, you can control multiple microphone channels, such as team and proximity chat. This controller is intended to solve that problem while also allowing you to push-to-talk or hold-to-talk on multiple channels without guessing if a chat channel is currently active. The mode switch on the bottom lets you use the same hardware for either keyboard or gamepad input, depending on the app's support for each HID type.

I also found this useful for controlling my WhisperTranscription application, as the dual buttons and tap-or-hold functionality are perfect for distinguishing between transcribing my text and issuing commands to an AI model. This is my current setup, though I am considering updating the hotkeys, as this application is far more flexible for using obscure keys or key combinations.

Overview


A custom USB device built using the nullbits Bit-C PRO. It has two buttons: one can be tapped to turn on or off, and the other is held down for push-to-talk. A switch changes between acting like a keyboard (Scroll Lock/Pause) and a gamepad (buttons 8-9). LEDs show the current state of each button. The game controller mode is configured directly using the AVR core's HID feature, without additional libraries.

The gamepad HID descriptor is implemented directly via the AVR core's HID.h — no third-party joystick library required.

Hardware

  • Board: nullbits Bit-C PRO (Arduino Leonardo-compatible)
  • Button 1 — D2, internal pull-up → GND
  • Button 2 — D3, internal pull-up → GND
  • LED 1 — D4 via 220 Ω resistor
  • LED 2 — D5 via 220 Ω resistor
  • Mode switch — D15 to GND = keyboard; floating = gamepad

Button Behaviour

  • Tap — toggle mic on/off (keyboard: key press+release; gamepad: brief button press)
  • Hold ≥ 50 ms — push-to-talk; latch active while held, cleared on release
  • Mode switch — releases all outputs and resets all latch states
  • Keyboard keys: Scroll Lock (btn 1), Pause (btn 2)
  • Gamepad buttons: 8 and 9 (beyond standard controller layouts)
