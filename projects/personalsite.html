View on GitHub

Overview

A hand-rolled personal site with no frameworks or build tools. Layout, theming, navigation, and blog functionality are all implemented in plain HTML, CSS, and vanilla JavaScript.

The nav panel is generated dynamically by a shared layout.js script injected on every page, keeping navigation consistent without a build step. A theme picker persists the selected colour scheme via localStorage.

Key Features

  • Dynamic nav and header injected via a shared layout script
  • Multi-theme colour picker persisted to localStorage
  • Blog with localStorage-backed post storage and admin panel
  • Mobile-responsive layout with hamburger nav drawer
  • No build step — deploy by copying static files
